
# Lovable.dev Frontend Prompt for SCARF

Project: SCARF (Scientific Claim-Assumption-Rationale Framework)
Domain: Scientific Reasoning & AI Document Analysis
Tech Stack: React, Tailwind CSS, Lucide Icons, Shadcn UI

## Core Goal
Build a modern, polished, and sophisticated dashboard for uploading research papers and visualizing their logical decomposition. The design should feel "scientific yet futuristic" (think Linear or Vercel style).

        *   Each card represents a **Scientific Claim**.
        *   **Badges**: Tags for claim confidence (e.g., "High Confidence").
        *   **Evidence Section**: An expandable accordion showing "Linked Evidence" from the paper.
        *   **Gap Alerts**: A warning block (Red/Yellow) if "Missing Evidence" or "Unverified Assumption" is detected.
        *   **Questions**: A green block suggesting "Research Questions" to validate the claim.

## API Integration Logic (Simulated for Generator)

*   `POST /upload`: Takes `file` form-data. Returns `{ job_id }`.
*   `GET /status/:id`: Returns `{ status: "PENDING" | "COMPLETED", progress: 0-100 }`.
*   `GET /report/:id`: Returns JSON with:
    ```json
    {
      "claims": { "claims": [ { "claim_id": "C1", "statement": "..." } ] },
      "gaps": { "analysis": [ { "claim_id": "C1", "signals": ["No baseline info"] } ] }
    }
    ```
